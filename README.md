# Constella · 考研词汇星系

一个为考研英语设计的**交互式词汇关系图谱**。每个单词是一颗星，词根词缀是星系中心，同义反义是连线——学单词的同时建造一座属于自己的词汇宇宙。

---

## 技术栈

| 层 | 技术 | 为什么选它 |
|---|---|---|
| 3D 渲染 | Three.js + React Three Fiber | 星空场景、Bloom 柔光、轨道公转 |
| 框架 | React 19 + TypeScript | 组件化 UI，类型安全 |
| 状态管理 | Zustand | 极简，比 Redux 轻 10 倍 |
| 构建 | Vite 8 | 秒级热更新 |
| 数据 | localStorage | 零配置，浏览器本地持久化 |

## 数据模型

**Word（单词）**：词条 + 多个释义 + 词根/前缀标记

**Relationship（关系）**：连接两个单词，类型包括：
- `synonym` 同义（绿）
- `antonym` 反义（红）
- `root-share` 词根（琥珀）
- `prefix-share` 前缀（紫）
- `suffix-share` 后缀（蓝）
- `derivative` 词族（橙）
- `similar-form` 形近（青）

每个关系可以绑定到具体的释义编号。

## 视觉设计

- **词根节点**：金色六角大星，星系中心
- **前缀节点**：紫色菱形星
- **普通单词**：彩色星芒，颜色=第一词性（动词粉/名词青/形容词黄/副词紫）
- **连线**：三层叠加贝塞尔弧线，颜色编码关系类型
- **背景**：暖色羊皮纸（`#f5f1e8`），长时间阅读不刺眼

### 为什么是羊皮纸？

暗黑星空 + Bloom 光晕在技术上极难调好——光晕容易脏、文字对比度差、长时间学习眼睛累。羊皮纸暖底 + 深色文字是经过多次迭代后最舒适的选择。

---

## 交互

- **默认态**：词根用 3D 黄金螺旋散射，子词沿倾斜轨道缓慢公转
- **点击一个词**：关联词被拉到中心围成圈，无关词淡化，镜头自动移到屏幕中央
- **再点一次**：打开右侧详情面板，编辑释义、增删关系
- **点空白处**：恢复全局视角

## 使用

**方式一（推荐）：** Finder 里双击 `start.command` → 自动构建并打开浏览器。

**方式二（终端）：**
```bash
cd constella
npm run dev        # 开发模式，热更新
npm run build      # 生产构建 + 启动静态服务
```

浏览器打开 `http://localhost:5173`。数据自动保存在 localStorage，刷新不丢失。导出按钮可备份 JSON。

## 开发历程

这个项目经历了五次视觉大改版：

1. **Cytoscape 2D 力导向** — 节点挤，无空间感
2. **Three.js 暗黑星空** — 光晕脏，文字不清
3. **AI 生图花卉花园** — 花抢眼，干扰记忆
4. **土星环星球** — 球太暗，不协调
5. **羊皮纸星图（当前）** — 暖底 + 星芒 + 弧线，干净舒适

每次迭代都基于实际学习体验反馈。代码记录了这一路走来的所有痕迹。

## 给开发者

本项目有详细的 [CLAUDE.md](./CLAUDE.md)，记录了所有设计决策、常见错误、交互规范。**在改任何代码之前先读它。**

关键约定：
- 不要换渲染引擎（Three.js 是目前的最优解）
- 不要换布局系统（确定性轨道布局比力仿真稳定）
- 不要用外部图片做节点（Canvas 程序化绘制最可靠）
- 改完代码必须跑 `npm run build` 确认零报错
